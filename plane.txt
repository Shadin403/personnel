aravel Personnel Management Admin Panel
Overview
Build a premium Laravel admin panel replicating the TestApplication from http://118.179.177.37 - a military/security personnel (soldier) management system. The original app is basic ASP.NET; we'll rebuild it as a Laravel app with premium Tailwind CSS UI.

Application Pages (from site analysis)
Page	Route	Description
Login	/login	Email + Password auth
Dashboard	/admin/dashboard	Overview stats
Soldier List (Admin Panel)	/admin/soldiers	Table with all soldiers
Enroll New	/admin/soldiers/create	User type dropdown + enroll
Update Profile	/admin/soldiers/{id}/edit	Full profile form
Download TRG	/admin/soldiers/{id}/download-trg	PDF download
Tech Stack
Framework: Laravel 11
UI: Tailwind CSS (CDN) + Alpine.js
Database: MySQL
Auth: Laravel Breeze/built-in
